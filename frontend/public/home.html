Network Analytics and Recommendation Systems for GitHub

Recommending projects to developers and developers to recruiters

We construct Collaboration Network of the projects on GitHub to recommend new projects to developers seeking out for them. We also construct a Collaboration Network of the active developers on GitHub to figure out similar developers.
